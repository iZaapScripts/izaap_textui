# izaap_textui
A modern and immersive Text UI system for FiveM servers.

## Features
- Modern Text UI
- Smooth animations
- Multiple prompts support
- Custom key and text
- Flexible positions
- Easy integration
- Optimized performance

## Exports
- `DrawText(text, key)`
- `ShowTextUI(data)`
- `HideText(id)`
- `UpdateText(text, id)`
- `UpdateKey(key, id)`
- `UpdateTextUI(data)`
- `IsTextUiOpen(id)`

## Support
- ESX
- QBCore
- Qbox
